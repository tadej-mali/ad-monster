Welcome to Advertisements Directory

Advertisements Directory is a sample app that demonstrates how to build web applications using different frameworks. This version is built with Backbone.js and Twitter Bootstrap.

Try It

Below there is a list of folders. Each folder contains a list of advertisements - you can see them by tapping on the folder description. For each advertisement there is a short description and a link to its preview. On tapping the description a new view shall open displaying the contents of the advertisement.

Get the Code

The source code for this application is maintained in this GitHub repository. The repository includes two adapters to use the application with different data persistence solutions: in-memory store and REST API.
